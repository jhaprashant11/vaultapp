# LocalVault

LocalVault is an Android password manager application built using Kotlin and Android Studio.  
The app allows users to securely store and manage passwords locally on their device with a clean and modern UI.

## Features

- Store passwords securely
- Modern Android UI
- Autofill service support
- Local storage based system
- Fast and lightweight
- Simple user experience

## Tech Stack

- Kotlin
- Android Studio
- Gradle
- Android SDK
